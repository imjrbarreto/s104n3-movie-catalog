# s104n3-movie-catalog
Movie Catalog
  
**Description**: A simple PHP poker dice app built to practice object-oriented programming concepts.

## Technologies

* PHP

## Installation and Execution

1. Clone the repository:

```bash
git clone [text](https://github.com/imjrbarreto/s104n3-movie-catalog)
```

2. Navigate to the project folder:

```bash
cd s104n3-movie-catalog
```

3. Start the PHP built-in terminal:

```bash
php index.php
```
